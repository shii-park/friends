# バトル
- シングルバトル
- オンラインバトル

## バトル報酬

- フリー石: 1個
- フリーコイン: 100コイン
- 勝ったら10RP
- 負けたら0RP

## 勝利条件
1枚vs1枚で、相手のHPを0にした方が勝利。

## フロー
### 準備フェーズ
1. 出すカード
2. 装備を決める

### 対戦フェーズ
1. じゃんけんの手を決める
2. じゃんけんの勝敗を決める
3. 勝ったほうの手が必殺技かどうかを判断
4. 必殺技だった場合 -> 必殺技を相手に攻撃する
4. 必殺技じゃなかった場合 -> 普通に攻撃
5. 体力が0以下になっているかをチェックする
6. なっていた時 -> 勝敗をつける
6. なっていなかった場合、1に戻る



## 攻撃

dmg = (ATK + BonusATK) * (0.8 ~ 1) + (TECH + BonusTECH) * (0.5 ~ 1) 

def = (ATK + BonusATK) * (0.3~0.5)

AfterHP = HP - ( dmg - def ) [dmg - def < 0 -> dmg - def = 0]
